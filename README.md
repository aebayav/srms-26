# SRMS-26 — InfraReport

A mobile application for infrastructure reporting, built with React Native and Expo.

## Project Structure

```
SRMS-26/
├── client-mobile/   # Expo/React Native mobile app
├── docker-compose.yml
└── .gitignore
```

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v20+)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)

### Install Dependencies

```bash
cd client-mobile
npm install
```

### Run the App

```bash
npx expo start
```

### Run with Docker

```bash
docker compose up --build
```

The web version will be available at `http://localhost:8081`.
